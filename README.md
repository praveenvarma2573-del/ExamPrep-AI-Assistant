# 📚 ExamPrep AI Assistant

## Project Overview

ExamPrep AI Assistant is an AI-powered chatbot developed using Python, Streamlit, and Groq LLM.

The chatbot helps students with:

* Exam Preparation
* Study Plans
* Quiz Generation
* Important Questions
* Interview Preparation
* Technical Learning
* Competitive Exam Guidance

## Features

### Exam Chatbot

Answers study-related questions in a student-friendly manner.

### Study Plan Generator

Creates study plans based on:

* 1 Hour
* 2 Hours
* Daily Plans
* Weekly Plans
* Monthly Plans

### Quiz Generator

Generates MCQ questions with answers.

### Important Questions

Provides important questions for exam preparation.

### Interview Preparation

Generates interview questions and answers.

## Technology Stack

* Python
* Streamlit
* Groq API
* Llama 3.3 70B Model
